## Kwik Trade Landing Page (React + Tailwind CSS)

## Description

A modern and responsive trading platform landing page built using React and Tailwind CSS.
This project replicates a professional trading website UI with features like OTP authentication UI, responsive design, and clean dark-themed interface.

## Features
🎥 Hero section with promotional video
💰 Attractive CTA (20% Bonus Banner)
📊 Highlight features section (0% brokerage, 500x margin, etc.)
📱 Fully responsive design (mobile + desktop)
🔐 OTP-based login UI (frontend)
🌍 Country code with flag in mobile input
🎨 Dark theme modern UI
📞 WhatsApp chat button integration
🧾 Registration form with validation UI

## Tech Stack
Frontend: React.js
Styling: Tailwind CSS
Languages: HTML, CSS, JavaScript
